# 📒 Contact Book App

## Project Overview

Contact Book App is a simple Python-based application that allows users to manage their contacts efficiently. Users can create, view, update, delete, search, and count contacts through a menu-driven interface.

## Features

* Add new contacts
* View existing contacts
* Update contact information
* Delete contacts
* Search contacts by name
* Count total contacts
* User-friendly menu interface

## Technologies Used

* Python 3
* Dictionary Data Structure
* Loops
* Conditional Statements
* User Input/Output

## Contact Information Stored

Each contact contains:

* Name
* Age
* Email Address
* Mobile Number

## How to Run

1. Install Python 3 on your system.
2. Save the program as `contact_book.py`.
3. Open Terminal/Command Prompt.
4. Run the following command:

```bash
python contact_book.py
```

##  Menu Options

1. Create Contact
2. View Contact
3. Update Contact
4. Delete Contact
5. Search Contact
6. Count Contact
7. Exit
